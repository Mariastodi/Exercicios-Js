## Estrutura do Repositório

O projeto está organizado em módulos para facilitar a navegação e o acompanhamento do aprendizado:

* **[Modulo 01](./modulo01):** Fundamentos - Variáveis, tipos de dados, operadores e estruturas condicionais.
* **[Modulo 02](./modulo02):** Estruturas de repetição, arrays e funções básicas.
* **[Modulo 03](./modulo03):** Manipulação de objetos, métodos de array e lógica intermediária.

---

## Tecnologias Utilizadas

* **JavaScript:** Linguagem principal.
* **Node.js:** Para execução dos scripts via terminal.
* **Git & GitHub:** Para versionamento de código.

---

## Como executar os exercícios

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório:
   ```bash
   git clone [https://github.com/Mariastodi/Exercicios-Js.git](https://github.com/Mariastodi/Exercicios-Js.git)
````

3.  Entre na pasta do módulo desejado:
    ```bash
    cd modulo01
    ```
4.  Execute o arquivo com o Node:
    ```bash
    node nome-do-arquivo.js
    ```
