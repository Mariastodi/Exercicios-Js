# Exercícios de JavaScript

Este repositório foi criado para armazenar meus estudos, desafios e exercícios práticos de **JavaScript**.
O objetivo é consolidar o conhecimento, desde a lógica básica até conceitos mais avançados da linguagem.

---

## Estrutura do Repositório

O projeto está organizado em módulos para facilitar a navegação e o acompanhamento do aprendizado.
Cada módulo representa uma etapa da evolução nos estudos, começando pelos fundamentos e avançando gradualmente para conceitos mais complexos.

```markdown id="modulos01"
### Módulos

* **[Módulo 01](./modulo01):** Fundamentos — variáveis, tipos de dados, operadores e estruturas condicionais.  
* **[Módulo 02](./modulo02):** Estruturas de repetição, arrays e funções básicas.  
* **[Módulo 03](./modulo03):** Manipulação de objetos, métodos de array e lógica intermediária.  
```

## Como executar os exercícios

Para rodar os exercícios localmente, siga os passos abaixo:

````markdown id="exec01"
### Passos para execução

1. Certifique-se de ter o Node.js instalado  
2. Clone o repositório:
   ```bash
   git clone https://github.com/Mariastodi/Exercicios-Js.git
````

3. Entre na pasta do módulo desejado:

   ```bash
   cd modulo01
   ```

4. Execute o arquivo com o Node:

   ```bash
   node nome-do-arquivo.js
   ```
